What is Lektor

If you are familiar with static site generators then the idea of lektor won't be a new one to you, but Lektor stands out because it emphasises on also being a content management system, CMS(s) are very popular even among non developers or programmers, because of their ease of use and accesibility, lektor tries to bring this accesibility and ease through its admin interface which makes managing the content of your static website, as easy and quick as if you were using a CMS like wordpress, for people who do not want to make a compromise between a CMS and a static blog engine lektor provides a middle ground.

Pre-requisites

To complete this tutorial you will need the following:

  • Python 3 installed with virtualenv and pip modules installed
  • a text editor (I like to use sublime text but any text editor you use is fine). no need for an IDE.
  • Git installed - you can install it here
  • an account on travis-ci.org (if you dpn't have one yet, go here and get it)
  • finially you will need a github account
